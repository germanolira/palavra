## Full Description (Máx 4000 caracteres)

```
Todo dia tem uma nova palavra escondida. Você tem seis chances para descobrir qual é.

Palavra é um jogo de adivinhação em português onde a lógica encontra o vocabulário. A cada 24 horas, uma palavra secreta de cinco letras é sorteada entre milhares de termos válidos da língua portuguesa. O desafio é sempre fresco: todo mundo recebe a mesma palavra no mesmo dia, então o campo está nivelado.

A mecânica é direta. Você digita uma palavra de cinco letras como palpite. O jogo responde com três cores. Verde quer dizer que a letra está no lugar certo. Amarelo indica que ela existe na palavra, mas em outra posição. Cinza elimina: aquela letra não entra na solução. Com essas pistas, você filtra letras, testa combinações e ajusta o rumo até acertar, ou até esgotar as seis tentativas. Cada partida dura poucos minutos, mas costuma viciar. A curiosidade de descobrir o termo do dia e a vontade de manter a sequência de vitórias fazem o jogo virar hábito.

Este é um passatempo para quem gosta de caça-palavras, palavras cruzadas, anagramas e desafios de raciocínio lógico. Também funciona como exercício mental para treinar memória, ortografia e pensamento dedutivo. Muitos jogadores usam como ritual matinal; outros, como descompressão no trajeto ou no intervalo do trabalho. Não depende de internet: uma vez instalado, roda 100% offline. Os palpites são validados localmente e o progresso fica salvo no dispositivo.

O visual é limpo e sem distrações. A interface prioriza o tabuleiro e o teclado, com feedback tátil opcional para cada tentativa. Há suporte a tema claro e escuro, ideal para jogar à noite sem cansar a vista. As animações são leves e rápidas, pensadas para não atrapalhar o ritmo.

Funcionalidades principais:
• Desafio diário com palavras em português
• Banco com milhares de palavras validadas de cinco letras
• Feedback visual por cores (verde, amarelo, cinza)
• Interface minimalista com tema claro e escuro
• Partidas rápidas, leves e 100% offline
• Salvamento automático do progresso
• Feedback tátil opcional
• Gratuito, sem anúncios intrusivos

Se você curte Wordle, Termo, Letreco ou qualquer desafio de palavras em português, Palavra traz a mesma fórmula que funciona, com o vocabulário local que faz sentido. Não há compras dentro do app nem necessidade de criar conta. Abra, jogue, descubra.

Baixe agora e veja se consegue adivinhar a palavra de hoje em menos de seis tentativas.
```

## Dicas de uso no Play Console

1. Cole a **Short Description** no campo "Descrição curta"
2. Cole a **Full Description** no campo "Sobre este app" (texto completo, incluindo bullets)
3. A descrição longa é indexada pelo Google — as keywords principais já estão distribuídas naturalmente ao longo do texto
4. A primeira frase (acima do "fold") é a mais importante para conversão
5. Os bullets com "•" deixam a descrição escaneável na loja
6. Não adicione keywords soltas no final — o Google penaliza stuffing
